# 星火計畫・排課中心

一個可直接部署在 GitHub Pages 的純前端排課系統，分成「用戶端」、「開課端」與獨立小遊戲頁面。

## 用戶端

- 顯示今日日期與最近即將到來的課程。
- 左側列出可選課程，支援搜尋與分類篩選。
- 課程時間會依開課端設定的時數顯示完整區段，例如 `19:00－20:30`。
- 優惠券區提供單一「點亮星火」按鈕前往小遊戲，不在每張課程卡重複顯示入口。
- 長按火堆 3 秒，光環與星火特效會依進度逐步展開。
- 完成後顯示「已點亮星火」，並隨機發放 40、60、80、100、120 或 150 點通用優惠券。
- 每個瀏覽器用戶只能成功點亮與領取一次；已領過舊版遊戲券的用戶也會自動視為已完成。
- 隨機優惠券具有不同名稱、稀有度、圖示與像素配色，並同步到排課中心的「我的優惠券」。
- 遊戲優惠券不指定課程，可套用於星火計畫中的任一課程。
- 可拖曳課程到中央「星火計畫區」，手機可按「加入」。
- 支援星火點數、固定折抵／百分比優惠券。
- 送出後自動扣除點數與名額，產生排課單總覽。
- 排課紀錄中的未開始課程可按「我要調課」。
- 調課只能置換同類型且仍有名額的課程。
- 每次調課固定收取 100 星火點數手續費；新課較貴時再加上新舊課程正差額，較便宜時不退點。
- 可使用名稱包含「調課」的優惠券，折抵調課手續費與課程差額的合計金額。
- 調課完成後會歸還原課程名額、扣除新課程名額並保存調課紀錄。
- 排課紀錄會保存在瀏覽器中。

## 開課端

- 建立單堂或批次課程，可設定日期、開始時間、課堂時數、老師、分類、點數與名額。
- 課堂時數支援 0.5 小時為單位，預設為 1.5 小時。
- 發放任意星火點數。
- 發放固定點數折抵或百分比優惠券，例如「調課卷」。
- 管理已發佈課程並查看操作紀錄。

## 技術

- HTML / CSS / Vanilla JavaScript
- 無需後端與外部套件
- 使用同網域的 `localStorage` 共享排課、一次性點亮狀態與遊戲優惠券資料
- 支援桌面、手機、滑鼠、觸控及鍵盤操作

> 此版本為單機展示版；「一個用戶一次」目前是以同一瀏覽器的本機資料判定。清除網站資料、使用無痕模式或更換裝置後會被視為不同用戶。若要跨裝置強制限制，需要登入系統與後端資料庫。
