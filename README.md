# 星火計畫・排課中心

一個可直接部署在 GitHub Pages 的純前端排課系統，分成「用戶端」與「開課端」兩個介面。

## 用戶端

- 顯示今日日期與最近即將到來的課程。
- 左側列出可選課程，支援搜尋與分類篩選。
- 課程時間會依開課端設定的時數顯示完整區段，例如 `19:00－20:30`。
- 可拖曳課程到中央「星火計畫區」，手機可按「加入」。
- 支援星火點數、固定折抵／百分比優惠券。
- 送出後自動扣除點數與名額，產生排課單總覽。
- 排課紀錄中的未開始課程可按「我要調課」。
- 調課只能置換同類型且仍有名額的課程。
- 每次調課固定收取 100 星火點數手續費；新課較貴時再加上新舊課程正差額，較便宜時不退點。
- 可使用名稱包含「調課」的優惠券，折抵調課手續費與課程差額的合計金額。
- 調課完成後會歸還原課程名額、扣除新課程名額並保存調課紀錄。
- 排課紀錄會保存在瀏覽器中。

## 開課端

- 建立單堂或批次課程，可設定日期、開始時間、課堂時數、老師、分類、點數與名額。
- 課堂時數支援 0.5 小時為單位，預設為 1.5 小時。
- 發放任意星火點數。
- 發放固定點數折抵或百分比優惠券，例如「調課卷」。
- 管理已發佈課程並查看操作紀錄。

## 技術

- HTML / CSS / Vanilla JavaScript
- 無需後端與外部套件
- 使用 `localStorage` 保存資料
- 支援桌面與手機版版面

> 此版本為單機展示版；不同裝置之間不會共享資料。若要讓開課端與所有用戶即時同步，需要再串接 Firebase、Supabase 或其他後端資料庫。
