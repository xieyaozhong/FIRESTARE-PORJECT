# 星火混合式柏青哥 V6.3

一款不需安裝套件、可直接在瀏覽器遊玩的原創柏青哥模擬遊戲，將物理彈珠、三軸老虎機和三層機械圓盤整合成同一套流程。

## 線上玩法流程

1. 長按蓄力，放開後發射彈珠。
2. 彈珠會沿封閉式金屬發射槽進入第一排撞針。
3. 彈珠撞擊釘子、彈力盤、旋轉撥輪、彈板與獎洞。
4. 命中 `START` 後啟動三軸老虎機。
5. 老虎機結果決定三層圓盤的 GO 洞數量、速度、通關率與獎勵。
6. 命中左上方 `GOLD LOCK` 可釋放得分 10 倍的黃金彈珠。
7. 連過三層取得最終 JACKPOT。

## 核心功能

- **FEVER 多球模式**：碰撞與得分累積能量，滿 100 後維持 12 秒，每次發射變成三球。
- **三種難度**：休閒、經典、極限；會影響重力、老虎機機率、圓盤通過率與獎勵倍率。
- **自動發射**：適合觀察物理碰撞與展示機台。
- **任務與成就系統**。
- **本機保存**：最高分、生涯統計、難度、音效與成就會保存在瀏覽器。
- **全螢幕與鍵盤操作**。
- **PWA 離線支援**。
- **GitHub Pages 自動部署工作流程**。

## V6 機械與黃金彈珠

- 左右旋轉撥輪、兩組自動彈力擋板、移動救球桿與側邊回流軌。
- 左上方 `GOLD LOCK` 是較難命中的特殊目標；擊中後會釋放一顆黃金彈珠。
- 黃金彈珠進入得分槽、START、拉霸即時獎勵與三層轉盤最終獎勵時，分數均為 10 倍。
- 777、鑽石、星星、BAR、鈴鐺、櫻桃、各種對子與特殊混合組合，會套用不同的轉盤 GO 洞數量、速度、通關率、獎金、額外彈珠或機械增壓效果。

## V6.1 防飛出修正

- 高速彈珠拆分成最多 6 個物理子步驟，避免單幀穿越牆體。
- 機台外圍加入不可見安全籠與最終位置回復機制。
- 極端碰撞速度會被限制，但正常彈力與機械互動仍保留。

## V6.3 100% 撞針入口布局

- 發射入口重新設計為封閉式金屬導流槽。
- 導流槽延續原有弧形發射軌，出口固定對準右側撞針區。
- 導流槽底部新增單向落球喉道與專用第一排撞針。
- 玩家發射的每一顆彈珠都會先通過撞針入口，再進入其他玩法區域。
- 若罕見碰撞造成彈珠停留在入口太久，隱藏式單向餵球閘會在槽內將彈珠送入撞針區。
- 右側旋轉撥輪下移，避免干擾入口並改善機台視覺層次。
- 防飛出安全邊界、黃金彈珠、FEVER 與拉霸轉盤連動皆保留。

## 操作

| 操作 | 手機／平板 | 電腦 |
|---|---|---|
| 蓄力發射 | 長按發射按鈕 | 按住空白鍵 |
| 自動發射 | 點擊按鈕 | `A` |
| 音效 | 點擊按鈕 | `M` |
| 全螢幕 | 點擊按鈕 | `F` |

## 本機執行

直接開啟 `index.html` 即可。Service Worker 離線功能需要透過 HTTP 伺服器執行，例如：

```bash
python -m http.server 8000
```

然後開啟 `http://localhost:8000`。

## GitHub Pages

專案內已包含 `.github/workflows/deploy-pages.yml`。在儲存庫設定中將 **Settings → Pages → Source** 設為 **GitHub Actions**，之後每次推送到 `main` 都會自動更新網站。
