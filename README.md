# 星火混合式柏青哥 V6.7｜三層轉盤核心機台

本版重新設計整台柏青哥外觀，同時完整保留盤面中央的三層轉盤系統。

## V6.7 機台重製

- 新增大型上方招牌與 `TRIPLE CRANK` 核心標誌。
- 左右加入獨立燈柱，使用紅、金、綠、藍燈號模擬實機 LED。
- 中央玻璃盤面加入金屬框、固定螺絲與玻璃反射。
- 下方新增實機式出珠托盤、風險模組與最終獎勵模組。
- 右側控制面板增加三層轉盤狀態示意。
- 桌面版保持機台與控制面板並排；手機版改為完整機台在上、控制面板在下。

## 完整保留功能

- 三層轉盤與每層綠色通關孔。
- START 拉霸與圖案組合連動。
- 黃金彈珠所有收益 ×10。
- 周圍彈性撞擊物。
- 旋轉撥輪、自動彈板與移動救球桿。
- FEVER 三球、自動發射、難度與任務系統。

## 操作

- 手機／平板：長按發射按鈕蓄力，放開發射。
- 電腦：空白鍵蓄力；`A` 自動發射；`M` 音效；`F` 全螢幕。
